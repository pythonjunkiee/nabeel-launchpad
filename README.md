# Nabeel Anwar Siddiqui — Portfolio

Personal portfolio website built with React, TypeScript, Tailwind CSS, and Vite.

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Vite
- shadcn-ui

## Run Locally
```sh
git clone https://github.com/pythonjunkiee/cyber-nabeel-launchpad.git
cd nabeel-launchpad
npm i
npm run dev
```
